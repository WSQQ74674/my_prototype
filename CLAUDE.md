# 软件原型：风险点库Web应用

## 目标

快速做最小可用原型(MVP)：前端页面+简单交互，无需后端数据库，纯前端静态原型、可直接运行演示

## 技术栈

HTML5 + Tailwind CSS + JavaScript，单页面、响应式

## 规范

1. 代码简洁、注释清晰、文件结构规范
2. 只做原型：UI界面、基础交互（增删待办）、不做复杂权限/持久化
3. 输出完整可直接打开的index.html，可直接浏览器打开运行
4. 每次修改先说明方案、再生成代码
